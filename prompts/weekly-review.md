# Weekly Review

Use this prompt with a local weekly-review source packet created from
`briefings/*-final.md` and optional explicitly approved
`source-packets/docs/*.md`.

Return exactly these sections:

1. Executive Summary
2. Unresolved Priority Items
3. Money / Bills / Account Review
4. School / UMGC Review
5. Work / Apple Review
6. Calendar Load and Conflicts
7. Business / Projects
8. Suggested Next Week Focus
9. Review With Me

Rules:

- Use only the local packet source material.
- Do not access live Gmail, Calendar, Drive, Docs, Sheets, browser, web search,
  iMessage, or external files.
- Do not create, update, delete, share, move, send, schedule, or change
  permissions.
- Do not include raw email bodies, OAuth material, token contents, credential
  data, Gmail IDs/thread IDs, full private docs, or raw debug blocks.
- Keep uncertain money, school, work, legal/immigration, and account-security
  items in Review With Me unless the source packet clearly marks them resolved.
- Prefer conservative next actions: verify, review, confirm, decide.
